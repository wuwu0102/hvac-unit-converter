# hvac-unit-converter V0.10
HVAC 專用單位轉換工具（中文優先、手機版優先）。

## V0.10 更新
- 頁面版本升級為 V0.10。
- 工程工具卡片簡化，保留：溫度、流量、壓力、流速、水管管徑建議、壓差估算流量（設備修正）、空調箱溫度粗估（AHU）。
- 「壓差估算流量（設備修正）」改為主輸入優先流程：先輸入目前壓差與管徑，先得到初估流量結果。
- 「進階設定：設備已知條件（可選）」移至卡片底部，提供參考流量與對應壓損作為可選修正基準。

## 支援功能
- 溫度
- 流量
- 壓力
- 流速
- 水管管徑建議
- 壓差估算流量（設備修正）
- 空調箱溫度粗估（AHU）
