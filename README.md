# hvac-unit-converter V0.7
HVAC 專用單位轉換工具。

## V0.7 更新
- 支援設備壓損修正流量計算。
- 使用 ΔP ∝ Q² 近似。

## V0.6 更新
- 壓差流量粗估改為顯示理想值與建議合理流量區間。
- 用常用流速範圍（小管 0.6~1.2 m/s、50A 以上 0.6~3.0 m/s）做初估判斷。

## V0.5 更新
- 介面全面中文化，並優化手機版卡片排版。
- 流速新增多單位換算（m/s、ft/s、mm/s、cm/s），一次顯示其餘單位結果。
- 水流量可直接換算並推薦水管管徑（LPM / L/s / m3/h / GPM）。
- 新增「壓差 + 管徑 -> 約略流量粗估」功能。

## 支援功能
- 溫度
- 風量
- 壓力
- 流速
- 水管管徑建議
- 壓差估算流量（設備修正）
