# hvac-unit-converter V0.4
HVAC 專用單位轉換工具。

V0.4 - 水管管徑建議修正與中文化顯示

## 支援單位
- Temperature（C / F）
- Airflow（CFM / CMH / m3/s / L/s / LPM / CMM）
- Pressure（Pa / kPa / mmAq / bar / psi / N/m2）
- Velocity（m/s / ft/s）

## 新增功能
- Water Pipe Size Suggestion 改為「水流量 -> 水管管徑建議」
- 顯示全面中文化，介面更直觀
- 建議結果直接顯示 A / inch / DN 對照
