# HVAC Unit Converter V0.13

HVAC 單位轉換工具（中文優先、手機版優先）。

## 功能列表
- 溫度轉換（C ↔ F）
- 流量轉換（CFM / CMH / m3/s / L/s / LPM / CMM）
- 壓力轉換（Pa / kPa / mmAq / bar / psi / N/m2）
- 流速轉換（m/s / ft/s / mm/s / cm/s）
- 流量對應管徑
- 壓差估算流量（設備修正）
- 電力單位換算（W / kW / MW / HP）
- 三相電力估算
- 單相電力估算
- 功率反推電流

## 電力模組說明
- 電力模組僅包含 W、kW、MW、HP。
- RT / BTU/h 屬於冷量，不列入電力模組。
